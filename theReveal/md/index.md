#### 组件库使用策略

##### 全量使用

使用所有组件，缺点：打包时会全部打包

```js

import onevue from 'onevue'

Vue.use(onevue);

```

##### 按需加载

只是用需要的组件，优点：可以最大化减少打包体积

```js

import {A, B, C} from 'onevue'

Vue.component(A.name, A);
Vue.component(B.name, B);

```
